# Resume Builder

A Flutter application for generating professional resumes in PDF format. 

## Features
- Form input for user details (Name, Email, Phone, Summary)
- Real-time PDF preview
- Export to PDF using `pdf` and `printing` packages

## Getting Started

1. Ensure Flutter is installed.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app.
